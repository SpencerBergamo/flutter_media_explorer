## Getting Started

Ready to dive in? To continue making this media explorer even better, follow these simple steps:

1. Clone the repository:
```bash
git clone https://github.com/[username]/flutter_media_explorer.git
```
2. Install Dependencies
```bash
flutter pub get
```
3. Run the App
```bash
flutter run
```
